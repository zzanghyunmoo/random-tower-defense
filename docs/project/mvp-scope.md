# MVP 범위

## 목표

하나의 스테이지에서 전투 시작, 타워 소환, 적 이동, 자동 공격, 웨이브 클리어 또는 패배까지 이어지는 최소 플레이 루프를 만든다.

## 핵심 루프

1. 플레이어가 전투를 시작한다.
2. 적이 웨이브 단위로 출현한다.
3. 플레이어가 재화를 사용해 랜덤 타워를 소환한다.
4. 타워가 범위 내 적을 자동 공격한다.
5. 적을 처치하면 재화를 얻는다.
6. 적이 끝 지점에 도달하면 플레이어 체력이 감소한다.
7. 모든 웨이브를 막으면 승리, 체력이 0이 되면 패배한다.

## 필수 기능

- 2D 전투 씬
- 적 경로 이동
- 웨이브 스포너
- 랜덤 타워 소환
- 타워 타겟팅과 발사체
- 적 체력과 사망 처리
- 골드 또는 소환 재화
- 플레이어 체력
- 승리/패배 상태
- 기본 HUD

## 완료 조건

- Android 해상도 기준으로 전투 화면이 깨지지 않는다.
- 에디터에서 Play를 누르면 한 판을 끝까지 플레이할 수 있다.
- 웨이브, 적, 타워의 주요 수치를 코드 수정 없이 데이터로 바꿀 수 있다.
- 승리와 패배가 명확하게 구분된다.
- 전투 중 콘솔 에러가 발생하지 않는다.
