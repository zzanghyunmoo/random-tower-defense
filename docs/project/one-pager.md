# 랜덤 타워 디펜스 원페이지 기획서

## 콘셉트

랜덤으로 소환되는 타워를 조합하고 강화하면서 웨이브를 버티는 2D 타워 디펜스 게임입니다. 모바일에서 짧게 반복 플레이할 수 있고, 나중에 Steam에서도 자연스럽게 플레이 가능한 구조를 목표로 합니다.

## 대상 플레이어

- 모바일에서 짧은 세션의 전략 게임을 즐기는 플레이어
- 운과 빌드 선택이 섞인 반복 플레이를 좋아하는 플레이어
- 랜덤 다이스, 랜덤 타워 디펜스, 오토 배틀러 계열의 성장감을 선호하는 플레이어

## 대상 플랫폼

- 1단계: Android
- 2단계: iOS
- 3단계: Steam

## 기술 스택

- Unity
- C#
- GitHub
- Notion
- Linear

## 디자인 원칙

- 즉시 이해되는 전투 흐름
- 랜덤성이 있지만 선택이 의미 있는 타워 조합
- 한 판이 짧고 재도전 욕구가 생기는 밸런스
- 모바일 조작을 우선한 단순한 입력
- 데이터 기반으로 빠르게 밸런스를 바꿀 수 있는 구조

## 초기 제외 범위

- 실시간 멀티플레이
- 서버 권위형 전투 판정
- 과금/광고 시스템
- 복잡한 계정 시스템
- 대규모 콘텐츠 제작
