# 의사결정 기록

## 2026-05-08: 게임 엔진과 언어

- 결정: Unity + C#
- 맥락: 2D 모바일 우선 개발, iOS/Android 배포, 장기적으로 Steam 출시 가능성
- 이유: Unity는 모바일 2D 게임 생태계와 배포 경로가 가장 안정적이고, C#은 게임 로직과 데이터 구조를 빠르게 만들기 좋다.

## 2026-05-08: 주 개발 운영체제

- 결정: macOS를 메인 개발 환경으로 사용하고 Windows는 Steam/Windows 검증용으로 사용한다.
- 맥락: iOS 빌드는 macOS와 Xcode가 필요하며, Android는 macOS와 Windows 모두 가능하다.
- 이유: macOS 하나로 Android/iOS 초기 검증을 모두 처리할 수 있다.

## 2026-05-08: 프로젝트 도구

- 결정: GitHub, Notion, Linear를 각각 형상관리, 위키, 이슈 관리에 사용한다.
- 맥락: 코드, 문서, 작업 큐의 책임을 분리한다.
- 이유: 개인 프로젝트라도 초반부터 흐름을 분리하면 나중에 기능과 아이디어가 늘어날 때 정리가 쉽다.
